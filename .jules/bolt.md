## 2024-06-15 - Parallelize Market Context Fetch and Gemini Search
**Learning:** In the `NewsAnalyzer`, fetching market context via Binance/Gemini Search and building the search query for Gemini's grounding analysis were previously done sequentially. However, Gemini's Google Search tool uses a prompt (`PromptKeys.NEWS_ANALYSIS_SEARCH_QUERY`) that only requires the `symbol` to search for news, not the full market context. The full market context is only needed for the LLM call that analyzes the results.
**Action:** Always identify which parts of an external call require what data. By making the `context` parameter in `analyzeNewsForSymbol` accept a `Promise`, we can initiate the `marketContext` fetch concurrently with Gemini's search query generation and Google Search network call, reducing overall latency.
